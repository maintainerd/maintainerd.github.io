# Maintainerd

[Official Website](https://maintainerd.github.io/)

Maintainerd is an open-source, self-hostable catalog of cloud-style product and platform services. Designed for developers and app teams, Maintainerd delivers modular services that can run independently or connect through **M9d-Core** for a shared control plane.

## What is Maintainerd?

Maintainerd provides a library of self-hosted cloud services for modern applications, including:

- **M9d Core** — tenant and service control plane
- **M9d Domains** — domain bindings, DNS automation, certificate lifecycle
- **M9d Docker** — Docker runtime provider
- **M9d Kubernetes** — Kubernetes runtime provider
- **M9d Database** — database provisioning and control
- **M9d Auth** — identity and access management
- **M9d Secret** — secrets management layer
- **M9d Gateway** — API edge and routing service
- **M9d Storage** — storage APIs and drive app
- **M9d Messaging** — notifications and communication
- **M9d CMS** — content and site publishing
- **M9d Workflow** — approvals and state machines
- **M9d Jobs** — queues and scheduled work
- **M9d Observability** — logs, metrics, traces, health
- **M9d Billing** — subscriptions and usage
- **M9d Project** — issues, boards, roadmaps
- **M9d Support** — tickets and feedback

## Why use Maintainerd?

Maintainerd is built for teams that want:

- full control over cloud services
- open-source deployment transparency
- self-hosted infrastructure without vendor lock-in
- a modular service catalog for platform and product needs
- a unified developer experience across independent services
