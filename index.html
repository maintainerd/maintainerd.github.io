Designed for flexibility.

Use Standalone

Need just an IAM or a Storage API? Spin up a single Docker container. Our services don't strictly depend on each other, giving you zero lock-in.

Run as an Ecosystem

Deploy M9d-Core to manage tenants, provision multiple instances of services (like Auth or Storage), and secure inter-service communication automatically.

// Approach 1: Standalone
docker run -d -p 3000:3000 maintainerd/auth
// Approach 2: Ecosystem Provisioning
const
core = new M9dCore({ provider: 'kubernetes' });

core.provision({
  tenant: 'acme-corp',
  services: ['auth', 'storage', 'workflow']
});
